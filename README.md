📝 To-Do List App

A simple To-Do List application built with HTML, CSS, and JavaScript.
Easily add tasks with a date and remove them when completed.

🚀 Features
	•	📌 View active tasks
	•	✅ Mark tasks as completed
	•	❌ Delete tasks
	•	🧹 Clear all completed tasks
	•	🧾 Clean and responsive UI

🛠️ Tech Stack
	•	HTML
	•	CSS
	•	JavaScript
